# 🎵 Spotify Pulse 2026: Global Streaming Insights

![Dashboard Preview](Screenshot%202026-05-01%20164927.png)

## 📊 Project Overview
This project is a comprehensive Data Analytics solution built entirely using **Power BI**. It provides a deep dive into global music streaming patterns, artist performance, and track longevity for the year 2026. The dashboard is designed with a high-end UI/UX focus, featuring a custom sidebar navigation for seamless data filtering.

## 🚀 Key Business Insights
Based on the analysis of the Spotify dataset, the following insights were uncovered:
* **Dominant Market Strategy:** **71.91%** of the streaming library consists of **'Evergreen' tracks**, proving that long-term musical value drives consistent engagement over temporary viral hits[cite: 1].
* **Artist Supremacy:** **BTS** stands as the most dominant artist in the global market, leading in total stream volume[cite: 1].
* **Engagement Velocity:** New tracks exhibit a higher **Average Viral Score** (~16.85M) compared to legacy tracks, indicating that fresh releases are essential for platform growth[cite: 1].
* **Market Sentiment:** The dashboard tracks **Stream Velocity**, currently showing a trend of **-271.99K**, helping identify shifting market interest[cite: 1].

## 🛠️ Technical Implementation
I utilized the full Power BI stack to ensure data accuracy and professional visualization:

* **Data Transformation (Power Query):** 
    * Structured raw streaming data and handled null values.
    * Categorized tracks into 'Evergreen', 'New', and 'Stable Hit' using conditional logic[cite: 1].
* **Analytical Modeling (DAX):** 
    * Developed measures for **Total Streams (373M)** and **Average Viral Score (14.53M)**[cite: 1].
    * Created dynamic time-series calculations for **Stream Velocity**[cite: 1].
* **UI/UX Design:**
    * **Custom Sidebar Pane:** Integrated a dedicated vertical sidebar for filters (Country, Genre, Longevity) to optimize canvas space[cite: 1].
    * **Visual Hierarchy:** Implemented a top-level KPI summary for immediate visibility of critical metrics.
